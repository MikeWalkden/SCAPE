Detailed Description

This is a probabalistic model of landsliding that is separate from SCAPE, runs after it. It takes the projections of cliff toe recession from SCAPE and uses them to predict cliff top location. It does this many times, randomly selecting sequences of landslide period and size. In this way an ensemble set of cliff top recession estimates is created. See Dawson, R., Dickson, M., Nichols, R., Hall, J., Walkden, M., Stansby, P., Mokrech, M., Richards, J., Zhou, J., Milligan, J., Jordan, A., Pearson, S., Rees, S., Bates, P., Koukoulas, S., Watkinson, A., (2009) Integrated analysis of risks of coastal flooding and cliff erosion under scenarios of long term change. Climatic Change, Climatic Change (2009) 95:249-288. DOI 10.1007/s10584-008-9532-8. Springer.
