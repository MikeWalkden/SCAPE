Detailed Description

This module of SCAPE contains a one-line beach model, with some additional algorithms.

A 'bar' is assumed to exist offshore. This does nothing except store sediment moved offshore from the beach. Sediment transport rates offshore during storms are determined by COSMOS output. A simple function has been assumed that moves material back during calm periods.
