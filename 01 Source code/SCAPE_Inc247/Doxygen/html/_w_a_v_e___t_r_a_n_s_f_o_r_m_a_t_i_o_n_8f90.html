module  wave_transformation_module
 The input wave conditions within the model are assumed to be constant throughout a tidal timestep. Wave transformation processes including; refraction and shoaling are also included within the model and are represented using linear wave theory (Kamphuis, 1992, Kamphuis, 2000).
